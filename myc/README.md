# NN 实验报告

- 主报告：`nn_experiment_report.html`
- 演示文稿：`nn_experiment_report.pptx`
- 逐字稿：`nn_experiment_report_script.md`
- 汇总表：`nn_experiment_summary.csv`
- 明细数据：`nn_experiment_summary.json`
- 图表资源：`assets/`
- 生成脚本：`generate_nn_report.py`
- PPT 生成脚本：`generate_nn_presentation.py`

在仓库根目录执行：

```bash
python myc/generate_nn_report.py
python myc/generate_nn_presentation.py
```

即可重新生成整份报告、PPT 和逐字稿。
